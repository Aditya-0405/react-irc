Git Commands:

git init: Initializes a Git repository in the current directory.
git clone <url>: Clones an existing Git repository from the given URL.
git add <filename>: Adds a file to the staging area for the next commit.
git add -A: Adds all modified files to the staging area for the next commit.
git commit -m "<message>": Commits the staged changes with a message.
git push origin master: Pushes the local commits to the remote repository on the "master" branch.
git pull origin master: Pulls the latest changes from the remote repository and merges them into the local branch.
git status: Shows the current status of the working directory and staging area.
git log: Shows the history of commits in the current branch.
git checkout <branch-name>: Switches to an existing branch.
git branch <branch-name>: Creates a new branch.
git merge <branch-name>: Merges the specified branch into the current branch.
git branch -d <branch-name>: Deletes a branch.
git remote add <name> <url>: Adds a new remote repository.
git remote -v: Shows the list of configured remote repositories.
git fetch <remote-name>: Fetches the latest changes from the specified remote repository.
git stash: Stash away your current working directory changes.
git stash pop: Revert the last stashed changes.
git revert <commit-hash>: Reverts a specific commit.
git diff <branch-name>: Shows the differences between the current branch and another branch.
